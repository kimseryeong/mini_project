### 3. 계산기 만들어보기
###### 24.03.10 - 12 (1차 기능구현 )
#### 📌 기능 구현 목록
- 간단한 사칙연산 계산기 기능
- 두 가지 방식으로 풀이
   1) eval() 함수를 사용하는 방식
   2) 수식 계산 알고리즘을 사용하는 방식 -> 입력받은 중위표현식을 후위표현식으로 변경 후 후위표현식을 계산하는 방식으로 진행 (스택 알고리즘)

⭐️ 계산 기능의 문제 발생
   - 두자리 수 이상의 숫자를 입력받는 경우 계산식에서 오류가 발생함
